About server

Kuudere's bunnyhop - is a small autobunnyhop server, there are 2142 zoned maps. We use shavit's bhoptimer and many other plugins that will be useful to you (check command section).

There will never be any garbage here like weapon skins, player models (possibly only for the replay bot), paid donations, music, and other unnecessary plugins.

Styles

  1. Normal - Basic autobhop style
  2. Sideways - Just sideways
  3. Half-Sideways - Jump with the W key held down
  4. W-Only - You can only use the W key
  5. Low Gravity - Just low gravity
  6. Segmented - You can save your time, speed and position using the menu
  7. Segmented LG - The same as LG, but with segments menu
  8. Scroll - Jump only on scroll
  9. 400 Velocity - Speed limit at 400 Velocity
  10. 600 Velocity - Speed limit at 600 Velocity
  11. Jump Impulse - High jump height
  12. Tool-Assisted - Autostrafe, segments, timescale and others
  13. Super TAS - The same as TAS, but less speed loss
  14. Tool-Assisted LG - The same as LG, but with tas
  15. Tool-Assisted JI - The same as JI, but with tas
  16. Auto-Strafe - Normal with autostrafing
  17. Seg Auto-Strafe - The same as Auto-Strafe, but with segments menu
  18. Stamina - Jump with stamina
  19. Hard Stamina - Jump with stamina, height lowered (0.9)
  20. Seg Hard Stamina - Hard Stamina with segments
  21. Prespeed - You can pick up more speed at the start
  22. Speedrun - You get more speed after each jump
  23. TAS Speedrun - The same as Speedrun, but with TAS menu
  24. Half-Life 2 - Experience close to HL2, scroll, Accelerated Back Hopping
  25. HL2 TAS - Half-Life 2 with TAS
  26. Unreal - You can push off the walls by shooting at them, you also have a double jump
  27. Surf HSW - Surf HSW
  28. Backwards - Just backwards
  29. S-Only - You can only use the S key
  30. A/D-Only - You can only use the A or D key

Commands

Timer

  1. !style - Choose your bhop style
  2. !r - Start/restart your timer
  3. !b - Start your timer on the bonus track
  4. !m - Start your timer on the main track
  5. !setstart - Set a specific respawn point
  6. !end - Teleport to endzone
  7. !pause - Toggle pause
  8. !savegame - Save or load a timer state
  9. !replay - Opens the replay bot menu
  10. !myreplay - Opens your replay menu
  11. !preview - Preview your unfinished replay
  12. !spec - Moves you to the spectators' team

HUD

  1. !hud - Opens the HUD settings menu
  2. !js - Opens the SSJ, JHUD and other settings menu
  3. !scrollhud - Toggle HUD for scroll
  4. !perf - Play sound for perfect jump in scroll
  5. !yawdiff - Show your yaw

Statistics

  1. !stats - Show the player's profile
  2. !rank - Show your or someone else's rank
  3. !top - Show the top 100 players
  4. !wr - View the leaderboard of a map
  5. !rr - View the recent #1 times set
  6. !sjwr - Global SourceJump database
  7. !sdbwr - Gloabal Strafe database
  8. !ssjtop - Show the top SSJ players

Maps

  1. !tier - Prints the map's tier to chat
  2. !nominate - Nominate map for !rtv
  3. !rtv - Vote for change the map
  4. !rate - Open map rating menu, see map rating, and vote on it
  5. !topmaps - See the highest rated maps on the server
  6. !worstmaps - See the lowest rated maps on the server

Helpers

  1. !noslide - Stops sliding on easybhop styles upon landing
  2. !line - Show the route of the WR
  3. !myline - Show the your route of the PB
  4. !edgehelper - Will show the edges of your hitbox if you stand on the edge
  5. !groundticks - Will show the time spent on the ground before jumping
  6. !jumpcircle - Will show the approximate distance of your jump
  7. !showclips - Show clips on map
  8. !showtriggers - Show triggers on map
  9. !sts - Open triggers settings
  10. !dist - Place two points and measure the distance between them
  11. !bst - Opens the board stats settings menu

Other

  1. !hide - Toggle players hiding
  2. !goto - Teleport to another player
  3. !noclipme - Toggle noclip
  4. !noclipspeed - Change speed of noclip
  5. !lj - Toggle LongJump stats
  6. !ljs - LongJump settings
  7. !lf - Enable/Disable Landfix
  8. !lfmenu - Open Landfix Menu
  9. !chatranks - Shows server ranks

Known Issues
